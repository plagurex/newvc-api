О сервере

New Valera Craft — это приватный майнкрафт сервер, созданный для ценителей технологических модов и спокойной игры в хорошей компании.

Приватность

Только по приглашениям. Своя система аккаунтов обеспечивает безопасность и комфорт.

Create - основа

Сборка модов построена вокруг Create — мода, добавляющего механизмы, кинетическую энергию и автоматизацию.

Свой лаунчер

Уникальный лаунчер для автоматической установки модов и обновлений.

Технологии

Дополнительные моды для удобства и расширения возможностей Create.

Статус сервера

Сервер в разработке

В данный момент проект еще не запущен. Мы активно работаем над созданием уникального игрового опыта.

Моды и сборка

Основные моды

Наша сборка основана на Fabric 1.20.1 с модом Create в качестве центрального элемента геймплея.

Create

Мод, добавляющий кинетические механизмы, автоматизацию и декоративные элементы.

Дополнительные аддоны

Create: Steam Powered, Create: Central Kitchen, Create: Crafts & Additions и другие.

Технологические моды

Refined Storage, Applied Energistics 2, Industrial Revolution и подобные.

Кастомизация и удобство

Моды для улучшения графики, оптимизации и добавления качества жизни.

Техническая информация

  • Версия Minecraft: 1.20.1
  • Загрузчик модов: Fabric
  • Основной мод: Create 0.5.1
  • Количество модов: ~60
  • Требования: 4+ GB RAM, видеокарта с поддержкой OpenGL 3.2

Наш лаунчер

Уникальный лаунчер New Valera Craft

Для игры на нашем сервере требуется специальный лаунчер, который автоматически установит все необходимые моды и настройки.

Своя система аккаунтов

Регистрация и вход через нашу систему, а не через аккаунты Mojang/Microsoft.

Автообновление

Лаунчер автоматически обновляет моды и конфигурации при каждом запуске.

Интеграция с сервером

Прямое подключение к серверу без необходимости вводить IP-адрес.

Регистрация

Станьте частью New Valera Craft

Для доступа к серверу необходима регистрация. Так как сервер приватный, регистрация возможна только по приглашениям.

1

Оставьте заявку

2

Ожидайте приглашение

Мы рассмотрим вашу заявку и вышлем приглашение, если появится свободное место.

3

Скачайте лаунчер

После получения приглашения вам будет доступен лаунчер для скачивания.
